Decription:
A very short sample/model of a instagram application. A "post" contains an image and the amount of "waste" associated with the photo.

Application Level:
A simple, but semi-complex project behind the scenes. Utilizes external packages such as Firebase, Firestore, Locations and Camera Hardare utilities.

Files:
The lib folder contains the sample code for the application, while .yaml file contains the required dependencies. 
In addition, there is a short video clip showing the application in action.
